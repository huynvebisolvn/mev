# artemis-template
An idiomatic template for Artemis MEV bots.

Run 
```bash
git clone https://github.com/mteam88/artemis-template <project name>
cd <project name>
rm -rf ./.git
git init <project name>
``` 
to start.

Be sure to fill out `main.rs` with your own code.
